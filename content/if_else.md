Иногда по мере выполнения кода нам нужно выполнять различные действия в зависимости от условий.

В JavaScript для этого можно использовать инструкцию `if` `else` так же называется ветвления

Инструкция `if(...)` проверяет является ли условие внутри скобок правдивым, то есть возвращает `true` 

Пример проверки возраста пользователя на 18+, если больше 18 пускаем дальше, если меньше, то останавливаем

```JavaScript
let age = 19;
if(age >= 18) {
    console.log("Ok");
} else {
    console.log("Error");
}
```

Еще один пример с блоком `else`

```JavaScript
let isSunny = true;

if (isSunny) {
    console.log("Идем гулять");
} else {
    console.log("Сидим дома");
}
```

Структура ветвлений всегда одинаковая, блок `else` выполняется только когда `if` возвращает `false`


```JavaScript
if (условие) {
    // код выполнится, если условие ИСТИНА
} else {
    // код выполнится, если условие ЛОЖЬ
}
```
