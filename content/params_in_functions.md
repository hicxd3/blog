В функциях в JavaScript мы можем передавать информацию используя параметры. 

Рассмотрим простой пример функции которая принимает 2 параметра `name` и `text` 

```JavaScript
function showMessage(name, text) {
	console.log(name + ': ' + text);
}

showMessage('Eva', 'Привет Шиноби!');
showMessage('Eva', 'Это я, твоя Ева');
```

При вызове функции, переданные ей значения копируются в локальные переменные `name` и `text` после чего используются в теле функции.
