Переменные нужны для хранения информации

Структура переменных состоит их следующих частей

Переменная создается с помощью ключевых слов `let` и `const` в старой версии JavaScript использовали `var` и его можно встретить в старом коде. 

Дальше идет имя переменной, например `name` 

Потом ставится знак присваивания = 

И идет значение переменной

Например 

```javascript
let name = Eva;
```

Отличие `let` и `const` заключается в том что значения let можно поменять в других конструкциях кода а значения const нельзя поменять, его используют для обозначения данных которые не нужно будет менять, например день рождения или имя.

Переменную можно просто объявить, а присвоить значение чуть позже по мере выполнения кода.

```JavaScript
let name;
name = Eva;
```

Если переменная объявлена через `let` мы можем переписать значение, например:

```JavaScript
let name = Eva;
name = Shinobi;
console.log(name); // Выведет Shinobi
```

### Имена переменных

При задании имен переменных нужно придерживаться нескольких простых правил.

Имена переменных могут содержать только буквы, цифры либо символы `$` и `_`  а так же первый символ не должен быть цифрой. Еще есть зарезервированные имена например `error` `let` `return` `class` `function` 

### Немного про `const`

Объявленная переменная через ключевое слово `const` которое содержит значения, эти значения нельзя изменить. 

Например:

```JavaScript
const dateOfBirthday = 27.06.2002
dateOfBirthday = 25.06.2002 // Выведет ошибку
```
