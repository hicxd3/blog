Переменные нужны для хранения информации

Структура переменных состоит их следующих частей

Переменная создается с помощью ключевых слов `let` и `const` в старой версии JavaScript использовали `var` и его можно встретить в старом коде. 

Дальше идет имя переменной, например `name` 

Потом ставится знак присваивания = 

И идет значение переменной

Например 

```javascript
let name = Eva
```

Отличие `let` и `const` заключается в том что значения let можно поменять в других конструкциях кода а значения const нельзя поменять, его используют для обозначения данных которые не нужно будет менять, например день рождения или имя.
