# Hello World in C

The `Hello, World!` program is one of the most basic programs you can write in any language.

```c
#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}
```

This simple code demonstrates how to display a message in the console using the C programming language.
